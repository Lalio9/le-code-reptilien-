Leçon 8 : Dictionnaires

1. Introduction aux Dictionnaires

Les dictionnaires en Python sont des structures de données qui stockent des paires clé-valeur. Chaque élément du dictionnaire a une clé unique, et vous pouvez y accéder en utilisant cette clé.

Les dictionnaires sont particulièrement utiles pour associer des informations liées entre elles, comme dans un carnet d'adresses ou une base de données simple.

2. Créer un Dictionnaire

Un dictionnaire est créé en utilisant des accolades {} et en séparant chaque paire clé-valeur par une virgule. La clé et la valeur sont séparées par un deux-points :.

# Exemple de dictionnaire
personne = {"nom": "Alice", "âge": 25}
print(personne["nom"])  # Affiche "Alice"

Dans cet exemple, le dictionnaire personne contient deux paires : "nom" et "âge".

3. Accéder aux Valeurs

Pour accéder à une valeur dans un dictionnaire, vous utilisez la clé entre crochets. Par exemple :

# Accéder à une valeur avec la clé
print(personne["âge"])  # Affiche 25

Si vous tentez d'accéder à une clé qui n'existe pas, Python lèvera une exception KeyError.

3.1 Utiliser la méthode get()

Une façon plus sûre d'accéder à une valeur est d'utiliser la méthode get(), qui retourne None si la clé n'existe pas, plutôt que de lever une exception :

# Utilisation de get()
print(personne.get("nom"))  # Affiche "Alice"
print(personne.get("adresse"))  # Affiche None

4. Ajouter ou Modifier des Éléments

Vous pouvez ajouter une nouvelle paire clé-valeur ou modifier une valeur existante en utilisant une clé :

# Ajouter ou modifier une valeur
personne["profession"] = "Développeur"
personne["âge"] = 26
print(personne)  # Affiche {"nom": "Alice", "âge": 26, "profession": "Développeur"}

5. Supprimer des Éléments

Vous pouvez supprimer une paire clé-valeur d'un dictionnaire de plusieurs manières :

  • del : pour supprimer une clé et sa valeur.
  • pop() : pour supprimer une clé spécifique et retourner sa valeur.
  • clear() : pour supprimer toutes les paires du dictionnaire.
# Supprimer une clé
del personne["profession"]
print(personne)  # Affiche {"nom": "Alice", "âge": 26}

# Utiliser pop pour supprimer et obtenir la valeur
âge = personne.pop("âge")
print(âge)  # Affiche 26
print(personne)  # Affiche {"nom": "Alice"}

# Supprimer toutes les paires
personne.clear()
print(personne)  # Affiche {}

6. Itérer sur un Dictionnaire

Vous pouvez parcourir les clés ou les valeurs d'un dictionnaire en utilisant une boucle for :

# Itérer sur les clés
for clé in personne:
    print(clé)

# Itérer sur les valeurs
for valeur in personne.values():
    print(valeur)

# Itérer sur les clés et les valeurs
for clé, valeur in personne.items():
    print(clé, valeur)
