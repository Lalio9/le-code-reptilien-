Leçon 5 : Boucles (for et while)

Introduction

Les boucles sont un outil fondamental en programmation. Elles permettent de répéter une séquence d'instructions jusqu'à ce qu'une condition soit remplie. En Python, deux types de boucles sont principalement utilisés : for et while.

1. La boucle for

La boucle for est utilisée pour itérer sur une séquence (comme une liste, une chaîne de caractères, ou une plage de nombres). Voici la syntaxe générale :

for variable in sequence:
    # Instructions à exécuter à chaque itération

Exemple :

# Afficher les éléments d'une liste
fruits = ["pomme", "banane", "cerise"]
for fruit in fruits:
    print(fruit)

1.1 Utilisation de range()

La fonction range() est très utile pour générer des séquences de nombres :

# Afficher les nombres de 0 à 4
for i in range(5):
    print(i)

# Afficher les nombres de 1 à 10
for i in range(1, 11):
    print(i)

1.2 Boucle imbriquée

Une boucle for peut être imbriquée dans une autre boucle :

# Afficher un tableau de multiplication
for i in range(1, 4):
    for j in range(1, 4):
        print(f"{i} x {j} = {i * j}")
    print("---")

2. La boucle while

La boucle while continue d'exécuter un bloc de code tant qu'une condition reste vraie. Voici la syntaxe générale :

while condition:
    # Instructions à exécuter tant que la condition est vraie

Exemple :

# Afficher les nombres de 1 à 5
i = 1
while i <= 5:
    print(i)
    i += 1

2.1 Boucles infinies

Une boucle while peut devenir infinie si la condition reste toujours vraie. Assurez-vous de définir une condition d'arrêt :

# Attention : boucle infinie (à éviter)
while True:
    print("Ceci est une boucle infinie.")

3. Instructions spéciales dans les boucles

Python propose des instructions spéciales pour gérer le comportement des boucles :

  • break : Permet de sortir prématurément d'une boucle.
  • continue : Saute à l'itération suivante sans exécuter les instructions restantes.
  • else : S'exécute si la boucle termine normalement (sans break).

Exemple :

# Utilisation de break
for i in range(10):
    if i == 5:
        break
    print(i)

# Utilisation de continue
for i in range(10):
    if i % 2 == 0:
        continue
    print(i)

# Utilisation de else
for i in range(5):
    print(i)
else:
    print("Boucle terminée.")
