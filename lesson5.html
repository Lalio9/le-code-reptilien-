Leçon 5 : Boucles (for et while)

Les boucles permettent d'exécuter un bloc de code plusieurs fois. Voici un exemple d'utilisation de la boucle `for` et `while` :

# Boucle for
for i in range(5):
    print(i)

# Boucle while
i = 0
while i < 5:
    print(i)
    i += 1
