Leçon 1 : Introduction à Python

Python est un langage de programmation populaire pour sa simplicité et sa lisibilité. Python permet aux développeurs de créer des programmes efficaces avec un minimum de code. Ce langage est interprété, ce qui signifie que vous pouvez exécuter vos scripts directement sans avoir besoin de les compiler au préalable.

Exemple de code :

# Votre premier programme Python
print("Bonjour, Python!")
