Leçon 4 : Conditions (if/else)

Les structures conditionnelles permettent d'exécuter des blocs de code uniquement si une condition donnée est vraie. Voici un exemple avec `if`, `elif`, et `else` :

# Exemple de condition
age = 20

if age < 18:
    print("Vous êtes mineur.")
elif age == 18:
    print("Vous êtes majeur, mais tout juste!")
else:
    print("Vous êtes majeur.")
