Leçon 4 : Conditions (if/else)

Introduction

Les structures conditionnelles sont essentielles en programmation. Elles permettent de contrôler l'exécution de blocs de code selon des critères spécifiques. En Python, les mots-clés if, elif, et else permettent de définir ces conditions.

1. Syntaxe de base

La syntaxe de base pour une condition if est la suivante :

if condition:
    # Instructions si la condition est vraie
elif autre_condition:
    # Instructions si l'autre condition est vraie
else:
    # Instructions si aucune condition n'est vraie

Chaque bloc d'instructions est délimité par une indentation (tabulation ou 4 espaces). Python repose sur cette indentation pour structurer le code.

2. Exemples pratiques

Voici un exemple simple :

age = 20

if age < 18:
    print("Vous êtes mineur.")
elif age == 18:
    print("Vous venez de devenir majeur!")
else:
    print("Vous êtes majeur.")

2.1 Conditions imbriquées

Les conditions peuvent être imbriquées pour gérer des cas complexes :

note = 85

if note >= 50:
    if note >= 75:
        print("Excellente performance !")
    else:
        print("Performance correcte.")
else:
    print("Insuffisant.")

2.2 Comparaisons multiples

Vous pouvez combiner plusieurs conditions avec les opérateurs logiques (and, or, not) :

temperature = 25

if temperature > 0 and temperature < 30:
    print("La température est agréable.")
else:
    print("La température est extrême.")

3. Cas pratiques

Voyons quelques exemples concrets :

3.1 Vérification de l'accès à un site

age = 16
autorisation_parentale = True

if age >= 18 or autorisation_parentale:
    print("Accès autorisé.")
else:
    print("Accès refusé.")

3.2 Catégorisation d'un nombre

nombre = -5

if nombre > 0:
    print("Le nombre est positif.")
elif nombre < 0:
    print("Le nombre est négatif.")
else:
    print("Le nombre est zéro.")
