Leçon 7 : Listes

1. Introduction aux Listes

Les listes sont des structures de données qui permettent de stocker plusieurs éléments dans une seule variable. Les éléments d'une liste peuvent être de différents types (nombres, chaînes de caractères, booléens, etc.) et peuvent être modifiés après leur création.

  • Les éléments d'une liste sont indexés à partir de 0.
  • Les listes peuvent contenir des éléments de différents types.
  • Les listes sont mutables, ce qui signifie que vous pouvez les modifier après leur création.

2. Créer une Liste

Voici un exemple simple pour créer une liste :

# Exemple de liste
fruits = ["pomme", "banane", "cerise"]
print(fruits)

Dans cet exemple, fruits est une liste contenant trois éléments : "pomme", "banane" et "cerise".

2.1 Accéder aux éléments d'une liste

Vous pouvez accéder à un élément d'une liste en utilisant son index. Rappel : l'index commence à 0.

# Accéder à un élément
print(fruits[0])  # Affiche "pomme"
print(fruits[1])  # Affiche "banane"

3. Ajouter des éléments dans une Liste

Les éléments peuvent être ajoutés à une liste avec la méthode append() :

fruits.append("orange")
print(fruits)  # Affiche ["pomme", "banane", "cerise", "orange"]

3.1 Ajouter plusieurs éléments à la fois

Vous pouvez aussi ajouter plusieurs éléments à une liste avec la méthode extend() :

fruits.extend(["kiwi", "mangue"])
print(fruits)  # Affiche ["pomme", "banane", "cerise", "orange", "kiwi", "mangue"]

3.2 Insérer un élément à une position spécifique

Utilisez la méthode insert() pour insérer un élément à une position précise dans la liste :

fruits.insert(1, "fraise")
print(fruits)  # Affiche ["pomme", "fraise", "banane", "cerise", "orange", "kiwi", "mangue"]

4. Supprimer des éléments d'une Liste

Vous pouvez supprimer des éléments d'une liste de différentes manières :

  • remove() : supprime le premier élément qui correspond à la valeur donnée.
  • pop() : supprime un élément à un index spécifique (ou le dernier élément si l'index est omis).
  • clear() : supprime tous les éléments de la liste.
# Supprimer un élément
fruits.remove("banane")
print(fruits)  # Affiche ["pomme", "fraise", "cerise", "orange", "kiwi", "mangue"]

# Supprimer l'élément à l'index 2
fruits.pop(2)
print(fruits)  # Affiche ["pomme", "fraise", "orange", "kiwi", "mangue"]

# Supprimer tous les éléments
fruits.clear()
print(fruits)  # Affiche []

5. Itérer sur une Liste

Les boucles sont souvent utilisées pour itérer sur les éléments d'une liste. Vous pouvez utiliser une boucle for pour accéder à chaque élément :

for fruit in fruits:
    print(fruit)

Ce code affiche chaque fruit de la liste.
