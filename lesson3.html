Leçon 3 : Opérations de Base

Introduction

Les opérations sont au cœur de la programmation. En Python, les opérateurs vous permettent de manipuler des données et de prendre des décisions. Cette leçon couvre :

  • Les opérateurs arithmétiques
  • Les opérateurs de comparaison
  • Les opérateurs logiques
  • Les opérateurs d'affectation

1. Opérateurs Arithmétiques

Ces opérateurs sont utilisés pour effectuer des calculs mathématiques :

  • + : Addition
  • - : Soustraction
  • * : Multiplication
  • / : Division
  • ** : Exponentiation
  • % : Modulo (reste de la division)
  • // : Division entière
# Exemples
a = 10
b = 3
print(a + b)   # Addition: 13
print(a - b)   # Soustraction: 7
print(a * b)   # Multiplication: 30
print(a / b)   # Division: 3.333...
print(a // b)  # Division entière: 3
print(a % b)   # Modulo: 1
print(a ** b)  # Exponentiation: 1000

2. Opérateurs de Comparaison

Ces opérateurs permettent de comparer deux valeurs et renvoient True ou False.

  • == : Égal à
  • != : Différent de
  • > : Supérieur à
  • < : Inférieur à
  • >= : Supérieur ou égal à
  • <= : Inférieur ou égal à
# Exemples
x = 5
y = 10
print(x == y)  # False
print(x != y)  # True
print(x > y)   # False
print(x < y)   # True
print(x >= 5)  # True
print(x <= 4)  # False

3. Opérateurs Logiques

Ces opérateurs permettent de combiner plusieurs conditions :

  • and : Renvoie True si toutes les conditions sont vraies.
  • or : Renvoie True si au moins une condition est vraie.
  • not : Inverse la valeur d'une condition.
# Exemples
a = True
b = False
print(a and b)  # False
print(a or b)   # True
print(not a)    # False

4. Opérateurs d'Affectation

Ils sont utilisés pour attribuer des valeurs à des variables, avec des variantes pour effectuer des calculs en même temps :

  • = : Affectation simple
  • += : Addition et affectation
  • -= : Soustraction et affectation
  • *= : Multiplication et affectation
  • /= : Division et affectation
# Exemples
x = 5
x += 3  # x = x + 3
print(x)  # 8
x *= 2  # x = x * 2
print(x)  # 16

Créez un programme qui utilise :

  • Un opérateur arithmétique (par exemple, addition ou multiplication).
  • Un opérateur de comparaison.
  • Un opérateur logique pour combiner plusieurs conditions.
