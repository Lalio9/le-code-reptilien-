Leçon 3 : Opérations de Base

Les opérateurs en Python permettent d'effectuer des calculs mathématiques, des comparaisons et des opérations logiques. Voici quelques exemples : 

# Opérations mathématiques
somme = 10 + 5
produit = 10 * 5
puissance = 2 ** 3

# Comparaisons
print(10 > 5)  # True
print(10 == 5)  # False

# Opérations logiques
print(True and False)  # False
print(True or False)  # True
