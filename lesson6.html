Leçon 6 : Fonctions

1. Qu'est-ce qu'une fonction ?

Une fonction est un bloc de code conçu pour effectuer une tâche spécifique. Les fonctions permettent de :

  • Réutiliser du code.
  • Rendre votre programme plus structuré et facile à lire.
  • Réduire les erreurs en évitant la duplication.

En Python, une fonction est définie avec le mot-clé def. Voici un exemple :

# Définir une fonction
def saluer():
    print("Bonjour tout le monde !")

# Appeler une fonction
saluer()

2. Passer des paramètres à une fonction

Les fonctions peuvent recevoir des paramètres pour personnaliser leur comportement :

# Fonction avec un paramètre
def saluer(nom):
    print(f"Bonjour, {nom} !")

saluer("Alice")
saluer("Bob")

Les paramètres permettent de rendre les fonctions plus flexibles.

2.1 Valeurs par défaut

Vous pouvez spécifier des valeurs par défaut pour les paramètres :

def saluer(nom="ami"):
    print(f"Bonjour, {nom} !")

saluer()          # Utilise la valeur par défaut
saluer("Alice")   # Remplace la valeur par défaut

3. Valeur de retour

Les fonctions peuvent renvoyer des valeurs grâce à l'instruction return :

def additionner(a, b):
    return a + b

somme = additionner(5, 3)
print(somme)  # Affiche 8

L'utilisation de return est essentielle pour obtenir des résultats exploitables.

4. Portée des variables

Les variables définies à l'intérieur d'une fonction sont locales à cette fonction :

def afficher_nombre():
    nombre = 10
    print(nombre)

afficher_nombre()
# print(nombre)  # Erreur : 'nombre' n'existe pas en dehors de la fonction

5. Fonction imbriquée et fonction lambda

5.1 Fonctions imbriquées

Une fonction peut être définie à l'intérieur d'une autre :

def fonction_principale():
    def fonction_interne():
        print("Bonjour depuis la fonction interne.")
    fonction_interne()

fonction_principale()

5.2 Fonction lambda

Les fonctions lambda sont des fonctions anonymes définies en une seule ligne :

addition = lambda a, b: a + b
print(addition(3, 5))  # Affiche 8
