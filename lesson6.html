Leçon 6 : Fonctions

Les fonctions sont des blocs de code que vous pouvez réutiliser. Vous pouvez définir des fonctions avec le mot-clé `def`. Exemple :

# Exemple de fonction
def saluer(nom):
    print("Bonjour, " + nom + " !")

saluer("Alice")
