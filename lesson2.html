Leçon 2 : Variables et Types

Les variables en Python sont utilisées pour stocker des données. Vous n'avez pas besoin de spécifier leur type (comme "int" ou "str"), car Python le détermine automatiquement. Voici un exemple :

# Exemple de variables
nom = "Alice"
age = 25
est_etudiant = True

print(nom, age, est_etudiant)
