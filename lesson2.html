Leçon 2 : Variables et Types

Introduction aux Variables

Une variable est un conteneur qui permet de stocker des informations. Ces informations peuvent être de différents types, comme du texte, des nombres ou des valeurs booléennes. En Python, il n'est pas nécessaire de déclarer le type de la variable à l'avance. Python le déduit automatiquement.

Création de Variables

Pour créer une variable, il suffit de lui donner un nom et de lui attribuer une valeur :

# Exemple simple de création de variable
nom = "Alice"
age = 30
est_etudiant = True

Règles pour les noms de variables

  • Les noms de variables doivent commencer par une lettre ou un underscore (_).
  • Ils ne doivent pas contenir d'espaces ni commencer par un chiffre.
  • Ils doivent être descriptifs pour faciliter la lecture du code.

Types de données en Python

Python prend en charge plusieurs types de données courants :

  • Chaînes de caractères (str) : pour stocker du texte.
  • Nombres entiers (int) : pour des nombres sans décimales.
  • Nombres flottants (float) : pour des nombres avec décimales.
  • Booléens (bool) : True ou False.
# Exemples de types de données
prenom = "Jean"  # Chaîne
age = 25         # Entier
poids = 70.5     # Flottant
est_adulte = True # Booléen

Fonction type()

Vous pouvez utiliser la fonction type() pour vérifier le type d'une variable :

# Vérification du type
print(type(prenom))  # 
print(type(age))     # 
print(type(poids))   # 
print(type(est_adulte))  #

Conversion entre types

Vous pouvez convertir une variable d'un type à un autre à l'aide de fonctions comme str(), int(), et float().

# Conversion
nombre = "42"
nombre_entier = int(nombre)  # Convertit "42" (str) en 42 (int)
texte = str(nombre_entier)   # Convertit 42 (int) en "42" (str)

Bonnes pratiques

  • Donnez des noms explicites aux variables.
  • Évitez les noms trop courts ou non significatifs (par exemple, a, b).
  • Écrivez le code de manière à ce qu'il soit lisible pour les autres développeurs.
